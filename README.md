## Project Overview

This webpage accepts a student name and marks for five subjects. When the form is submitted, JavaScript calculates the total marks, average percentage, and grade, then displays the result dynamically in the webpage.

## ES6 Concepts Used

| ES6 Feature | Implementation |
| --- | --- |
| `let` and `const` declarations | `const` stores DOM references and fixed data, while `let` updates the grade decision |
| Arrow functions | Calculation, grading, rendering, and event callbacks use arrow function syntax |
| Template literals | The result card is created with template literal HTML |
| Array methods | `map`, `reduce`, and `some` process the subject marks |
| Object shorthand | Result data is passed between functions as an object |

## Features

| Requirement | Implementation |
| --- | --- |
| Enter student details | Student name and five subject mark inputs |
| Calculate total marks | Adds all five subject marks |
| Calculate average | Divides total by number of subjects |
| Assign grade | Uses average percentage to return A+, A, B, C, D, or F |
| Dynamic output | Updates the result panel without reloading the page |
